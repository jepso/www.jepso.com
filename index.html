Free Development for non-profit organisations

JEPSO have supported a number of non-profit organisatioins by developing and hosting free web based applications. These provide a great opportunity for JEPSO to give something back to the community, and to showcase the work we are so proud of.

Shepperton Slalom Canoe Club used to keep all their records on paper. This made it difficult to keep track of who had paid the appropriate membership fees and signed the correct disclaimers. It also made it impossible for them to compile lists of who had what qualifications or to produce statistics showing how well members were doing competitively. JEPSO revolutionised membership by providing a modern online database to store all membership information. This can be easily kept up to date by members. It allows coaches to access information on smart phones, as well as on desktop computers and makes it easy for the membership secretary to keep track of who has joined or left the club.

The RCSA (the student body for Robinson College at the University of Cambridge) had lots of separate systems for keeping track of both who had chosen what room, and descriptions/reviews of the various rooms. This made it stressful for the person responsible for running the room ballot and difficult for students to make informed decisions about which room they want. JEPSO took all the information stored across the various systems and combined them into one unified database. Having done this, we were able to create a modern, fast JavaScript front end. We used the universities authentication system so that students don't need to remember an aditionial username or password. We also added real-time capabilities, so that during the ballot students are always kept completely up to date as to which rooms are already taken vs. which rooms are still available.

The RCSA also had an old, outdated website running an old version of Drupal. This old system was prone to frequent crashes and had confusing bugs when attempting to edit content. This meant that all editing of content had to be performed by the administrator. Security was an all or nothing affair, so it was not possible to restrict any content to university members only. JEPSO created a new website using the latest version of Drupal and creating a totally customised theme. This not only updated the style of the website, but also offered speed improvements. The new website is carefully configured to be highly modular and maintainable. It uses the universities central authentication system so users don't need to learn a new username or password. Permissions can be defined in a fine grained fassion so that each role within the committee is only given access to edit content relevant to them, making it imposible for non-admin users to break the website. We also enforced consistent styling rules to provide the website with a consistent and professional image.
